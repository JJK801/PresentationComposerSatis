Orchestration

Utiliser Packagist

composer.json:

{
    "repositories": [],
    "require": {
        "acme/my-public-packagist-dependency": "dev-master"
    }
}

Construire Satis

satis.json:

{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://my-repo.tld/my-private-satis-dependency.git"
        }
    ],
    "require": {
        "acme/my-private-satis-dependency": "*"
    }
}
lancement du build:

$ ./bin/satis build ./satis.json /build
mettre à jour un package:

$ ./bin/satis build ./satis.json /build acme/my-private-satis-dependency

Utiliser Satis

composer.json:

{
    "repositories": [
        {
            "type": "composer",
            "url": "http://my-satis-domain.tld"
        }
    ],
    "require": {
        "acme/my-private-satis-dependency": "dev-master"
    }
}

Utiliser Toran Proxy

composer.json:

{
    "repositories": [
        {
            "type": "composer",
            "url": "http://user:pass@my-toran-domain.tld"
        }
    ],
    "require": {
        "acme/my-public-packagist-dependency": "dev-master"
    }
}

Désactiver Packagist

composer.json:

{
    "repositories": [
        {
            "packagist": false
        }
    ]
}

Etape 1:

Comment on gère nos dépendances internes ?

Besoins:

  • Un référentiel des projets internes (packages.json)
  • Des archives compressées (dist) des versions stables
  • Un build automatisé
  • Des notifications sur l'état des build

Architecture

Configuration Satis

satis.json:

{
    "name": "My amazing new Satis",
    "homepage": "http://satis.domain.tld/",
    "archive": {
        "directory": "dist",
        "format":    "zip",
        "skip-dev":  true
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://my-repo.tld/my-private-satis-dependency.git"
        },
        {
            "type": "vcs",
            "url": "https://my-repo.tld/my-private-satis-dependency-2.git"
        }
    ],
    "require-all": true
}

Configuration Projet

composer.json:

{
    "repositories": [
        {
            "type": "composer",
            "url": "http://satis.domain.tld"
        }
    ],
    "require": {
        "acme/my-private-satis-dependency":   "~2.3",
        "acme/my-private-satis-dependency-2": "dev-master",
        "twig/twig":                          "~1.7"
    }
